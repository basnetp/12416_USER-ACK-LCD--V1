

Version 1.0.0
Sept 9, 2019
Compiler: XC16 1.34
Environment: MPLABX 5.10
Hardware: ASM002865A

Description:
Demo software that initializes and exercises the hardware.
* the horn is chirped briefly
* A color palette and RLE-encoded image is displayed
* The external flash is checked
* Display backlight is cycled off then back on
* Then, the digital outputs start toggling once per second
* Status info received from the CS1108 is shown on the display.
* UART interface is available for debug 'printf' outputs
